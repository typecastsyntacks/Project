# My Ugly Website 🎨 → ✨

Welcome to **SMG Bubble — Week 1**!

This folder contains a personal website that looks... pretty bad on purpose.
Your mission today: **make it yours, and make it beautiful — using Claude Code.**

You won't write CSS or JavaScript by hand. You'll *direct* Claude Code with
clear instructions and watch the site transform.

---

## 1. See the ugly website first

Double-click `index.html` — it opens right in your web browser. That's it.
Witness the chaos. 😅

> Want it to auto-refresh when you make changes? In VS Code, install the
> **Live Server** extension, then right-click `index.html` → "Open with Live
> Server." (Optional — opening the file directly works fine for today.)

---

## 2. The files

| File          | What it is                                                        |
|---------------|-------------------------------------------------------------------|
| `index.html`  | The **content** — your text, sections, and structure.             |
| `styles.css`  | The **look** — colors, fonts, spacing, sizes. (This is the ugly part.) |
| `README.md`   | This file.                                                        |

You don't need to understand the code. That's the whole point of vibe coding.

---

## 3. Your job

There are two parts:

**A) Make it about YOU.**
Search the files for text inside `[[ double brackets ]]` — those are
placeholders. Replace them with your real info: your name, your bio, your
interests, your projects, your contact. You can ask Claude to do this for you.

**B) Make it look good.**
The colors clash, the fonts fight, nothing lines up, and the photos are tilted
at weird angles. Fix it — one request at a time.

---

## 4. Prompts to try with Claude Code

Start small. Change one thing, look at the result, then ask for the next thing.

```
Make this website look clean, modern, and easy to read.
```
```
The colors clash badly. Pick a calm color palette and apply it to the whole site.
```
```
The fonts don't go together. Choose two fonts that pair well and use them everywhere.
```
```
Fix the spacing and alignment so sections line up and have room to breathe.
```
```
The photo boxes are stretched and tilted. Make them neat, equal-sized, and straight.
```
```
Replace the placeholder text with info about me: my name is ___, I'm a ___ grader
from ___, I love ___, ___, and ___, and I once built ___.
```
```
Make this look great on a phone too.
```

**The secret:** be specific, change one thing at a time, and react to what you
see. If you don't like a result, just tell Claude what to do differently.
